# 数据结论摘要表

## 1. 关键数据结论

| 项目 | DWT | CAT | 结论 |
|---|---:|---:|---|
| 最优宿主图 PSNR 对应 alpha | 2 | 2 | 两方案在最小嵌入强度下不可见性最好 |
| `alpha=2` 时 PSNR/dB | 46.8089 | 46.8089 | 初始视觉质量都很高 |
| `alpha=10` 时 PSNR/dB | 32.8295 | 32.8296 | 高强度嵌入下视觉失真明显上升 |
| 平衡型 alpha（PSNR≥35 dB 时平均 NC 最优） | 6 | 6 | 两方案都在 `alpha=6` 左右达到较好折中 |
| 最强综合鲁棒性 alpha | 10 | 10 | 大 `alpha` 有助于提高攻击后提取稳定性 |
| `alpha=10` 平均攻击后 NC | 0.7965 | 0.8302 | CAT 的综合鲁棒性更强 |

## 3. 最重要的结论提炼

1. 两种方案在无攻击条件下都能近乎无误提取水印，因此比较重点不在“能否提取”，而在“受到攻击后谁更稳定”。
2. DWT 是 JPEG 友好型方案，适合强调压缩鲁棒性。
3. CAT 是综合稳定型方案，在当前实验数据上对椒盐噪声和中值滤波更稳健。
4. 若强调不可见性中的结构保真性，CAT 更优，因为其 `SSIM` 在所有 `alpha` 下都高于 DWT。
5. 若需要在课程作业中寻找“不可见性与鲁棒性的折中点”，`alpha=6` 是最合适的代表性参数。
6. 两种方案对裁剪缩放都较敏感，说明当前实现尚不具备强几何攻击鲁棒性。

## 4. 写入报告时可直接引用的结论句

1. 在当前默认实验数据上，CAT 方案在 `PSNR` 基本不变的前提下取得了更高的 `SSIM`，说明其对图像结构的扰动更小。
2. DWT 在三种 JPEG 压缩条件下都优于 CAT，表明其更适合应对压缩类失真。
3. CAT 在椒盐噪声和中值滤波条件下显著优于 DWT，说明四分组重排方式对随机脉冲和局部平滑攻击具有更强适应性。
4. 从课程题目契合度看，CAT 方案最能体现 “four groups for transform domain” 的设计思想；从经典频域水印角度看，DWT 更具教材标准性。
