# 课程作业报告摘要版结论

## 核心结论

1. 本实验完整实现了两套数字图像水印系统：DWT 频域方案和 CAT 风格四分组变换域方案，均支持 `512×512` 彩色宿主图像和 `256×256` 水印图像的嵌入与非盲提取。
2. 两种方案在无攻击条件下都能够实现近乎无误的水印恢复，`NC` 基本达到 `1`，说明当前嵌入与提取流程正确、稳定。
3. 在不可见性方面，两种方案的 `PSNR` 与 `MSE` 基本一致，但 CAT 在所有 `alpha` 下的 `SSIM` 都高于 DWT，说明其在当前实验图像上的结构保真性更好。
4. 当 `alpha` 从 `2` 增大到 `10` 时，宿主图像 `PSNR` 从约 `46.81 dB` 下降到约 `32.83 dB`，体现了嵌入强度增大带来的典型“不可见性下降、鲁棒性上升”关系。
5. 在 JPEG 压缩攻击下，DWT 的表现优于 CAT，尤其在 `JPEG_Q50`、`JPEG_Q70`、`JPEG_Q90` 三种压缩条件下都保持更高的平均 `NC`。
6. 在椒盐噪声和中值滤波攻击下，CAT 明显优于 DWT，其中中值滤波是 DWT 的最弱项，而 CAT 对此类平滑型攻击更稳定。
7. 在综合攻击平均 `NC` 上，CAT 在所有 `alpha` 下都高于 DWT；在当前默认实验数据中，`alpha=10` 时 CAT 的平均攻击后 `NC` 为 `0.8302`，高于 DWT 的 `0.7965`。
8. 若强调经典频域方法与 JPEG 鲁棒性，可优先选择 DWT；若强调课程题目中的 “CAT four groups” 思想、结构保真性与综合攻击稳定性，则 CAT 更适合作为重点比较方案。

## 摘要式表述

1. 本文不仅完成了方法实现，还对不可见性、无攻击提取和多种攻击鲁棒性进行了完整量化分析。
2. DWT 与 CAT 并非简单的“谁绝对更强”，而是在不同攻击类型下各有优势，这一点是本实验最重要的分析结论。
3. 对课程作业而言，CAT 方案既符合题目中的 four groups 描述，又在当前数据上表现出较好的综合性能，可作为报告中的重点比较方案。
